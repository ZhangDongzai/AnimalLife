# 宠物养成 Animal Life

这是一个 *终端* 的 **应用程序**, 玩家可以通过文字打怪闯关升级.

## 背景

在使用终端的时候, 我对纯文本的界面产生了丰厚的兴趣, 所以便有想法写一个这样的游戏.

## 使用

### 依赖
- 需要安装`python3`及以上版本

### 运行
在终端中运行以下代码

    git clone https://github.com/ZhangDongzai/AnimalLife.git; cd AnimalLife; python3 main.py
